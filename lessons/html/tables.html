Tables

Tables allow web developers to organise data neatly into rows and columns.

Example

Day Date Time Location
Monday 16/05/2022 3:20pm School
Tuesday 17/05/2022 5:30pm Home
Wednesday 18/05/2022 7:20am Friend's House

Define a HTML Table

A table in HTML consists of table cells inside rows and columns

Example

<!DOCTYPE>
<html>
  <head>
    <Page Title>
  </head>
  <body>
    <table>
      <tr>
        <th>Day</th>
        <th>Date</th>
        <th>Time</th>
        <th>Location</th>
      </tr>
      <tr>
        <td>Monday</td>
        <td>16/05/2022</td>
        <td>3:20pm</td>
        <td>School</td>
      </tr>
      <tr>
        <td>Tuesday</td>
        <td>17/05/2022</td>
        <td>5:30pm</td>
        <td>Home</td>
      </tr>
      <tr>
        <td>Wednesday</td>
        <td>18/05/2022</td>
        <td>7:20am</td>
        <td>Friend's House</td>
      </tr>
    </table>
  </body>
</html>

Table Cells

Each table cell is defined by the <td> and </td> tag. The <td> tag stands for 'table data'.

Everything between the <td> and </td> is the content of the table cell.

Example

<table>
  <tr>
    <td>Jun</td>
    <td>Mark</td>
    <td>Sanjana</td>
  </tr>
</table>

Note: table data elements are the data containers of the table. They can contain all sorts of HTML elements; text, images, lists, other tables, etc.

Table Rows

Each table row starts with the <tr> and ends with the </tr> tag. The <tr> tag stands for 'table row'.

Examples

<table>
  <tr>
    <td>Jun</td>
    <td>Mark</td>
    <td>Sanjana</td>
  </tr>
  <tr>
    <td>1.73m</td>
    <td>1.47m</td>
    <td>1.64m</td>
  </tr>
<table>

You can have as many rows as you like in a table, just make sure that the number of cells are the same in each row.

Table Header

Sometimes you want your table cells to have a header, in those cases, we use a <th> tag instead of a <td>.

Example

<table>
  <tr>
    <th>Person 1</th>
    <th>Person 2</th>
    <th>Person 3</th>
  </tr>
  <tr>
    <td>Jun</td>
    <td>Mark</td>
    <td>Sanjana</td>
  </tr>
  <tr>
    <td>1.73m</td>
    <td>1.47m</td>
    <td>1.64m</td>
  </tr>
<table>

By default, the text in <th> elements are bold and centered, but you can change that with CSS.
