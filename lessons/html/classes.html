The ID and Class attribute

The id attribute specifies a unique id for an HTML element. The value of the id attribute must be unique within the HTML document.

The id attribute is used to point to a specific style declaration in a style sheet. It is also used by JavaScript to access and manipulate the element with the specific id. To include an id, you put the id in whatever tag, insert quotes and give the id a name. For example <h1 id="myfirstid">Hello, World!</h1>

In the following example we have an <h1> element that points to the id name "myHeader". This <h1> element will be styled according to the #myHeader style definition in the head section:

Example:

<!DOCTYPE html>
<html>
  <head>
    <style>
      #myFirstID{
        text-align: center;
        color: black;
      }
    </style>
  </head>
  <body>
    <h1 id="myFirstID">My Header</h1>
  </body>
</html>

Difference Between Class and ID

A class name can be used by multiple HTML elements, while an id name must only be used by one HTML element within the page.

Instead of using a # before to select the class use a . instead also change the id="" to class=""

Example:

<!DOCTYPE html>
<html>
  <head>
      <style>
         .Class{
           text-align: center;
           color: black;
         }
      </style>
  </head>
  <body>
    <h1 class="Class">My Header</h1>
  </body>
</html>
