HTML Text Formatting

HTML contains several different elements for defining text.

Example

<em>italics</em>
  • <b> Bold Text
  • <i> Italic Text
  • <mark> Highlighted Text
  • <small> Smaller Text
  • <del> Deleted Text
  • <ins> Underlined Text
