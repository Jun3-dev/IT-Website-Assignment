How should you structure your website?

  1. index.html This file will generally contain your homepage content, that is, the text and images that people see when they first go to your site.
  1. Images Folder: This folder will contain all the images that you use on your site.
  1. Styles Folder: This folder will contain the CSS to style your content

Paths

To make files talk to one another, you have to provide a file path between them

  • First, drag and drop the images you want to display on your webpage into the images folder
  • Open up your index.html file, and insert the following code: <img src="" alt="my image">
  • Between the quotation marks of src = "" put the path of your file e.g images/myimage.jpeg, currently our image is in the "image" directory
  • If your file is outside of the current folder, you can put "../" at the start of the PATH to signal the computer to exit the current folder.
  • Save your html and then click the "Go Live" button on your Visual Studio Code IDE.
  • You should now see your image that you wanted to display on the website
