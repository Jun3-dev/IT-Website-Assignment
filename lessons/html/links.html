HTML Links and Multi-page Websites

Links are found in nearly all web pages. Links allow users to click their way from page to page.

HTML Links - Hyperlinks

  • HTML Links are hyperlinks
  • You can click on a link to be directed to a new document or webpage
  • When you move the mouse over a link, the mouse arrow will turn into a little hand.

Note: Links don't have to only be text. They can be disguised as images and any other HTML element!

HTMl Links - Syntax

The HTML <a> tag defines a hyperlink. It has the following syntax:

<a href="url">link text</a>

The most important attribute of the <a> element is the href attribute which defines the link's destination. The link text is the part that is visible to the reader. Clicking on the linked text will direct the reader to the specified URL address.

Example

<a href="www.facezongoogappflix.com/">Visit facezongoogappflix</a>

By browser and HTML default, links are appeared as following in browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red

You can style links to appear and look differently using CSS.

Multi-page Websites

To make a multipage website, simply make a new file in the same project and name it differently to the page you have made already. In this example, I am going to use index.html and make my second page australia.html. This is how you make a multi-page website. Now we are going to make a link that so when you click on it, it goes to your second page.

Syntax

<a href="page1.html">Next page</a>

See how I put the page name of the page I wanted to go to, this is what you have to do to make sure that it goes to the next page, otherwise it will come up with File not Found
