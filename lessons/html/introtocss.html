Introduction to CSS

CSS Is the language which we use to syle a webpage

What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Why use CSS

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Example

body {
  background-color: lightblue;
}
h1 {
  color: white;
  text-align: center;
}
p {
  font-family: verdana;
  font-size: 20px;
}

CSS Solved a Big Problem

HTML was never intended to contain tags for formatting a web page. HTMl was created to describe the content of a web page and provide information on it. For example:

<h1>This is a heading </h1>
<h2>This is a paragraph </p>

When tags like <font> and colour attributes were added to HTML it started causing frustration amongst web developers. To solve the problem, the World Wide Web Consortium (W3C) created CSS. CSS removed the style formatting from the HTMl page.

CSS Saves a Lot of Work

The style definitions are normally saved in external .css files. With an external stylesheet file, you can change the look of an entire website by changing just one file.

How to include CSS in a Page

To include CSS in a page, make a new file the same way you make HTML files and name it "style.css". Next, you need to link the css to the HTMl page otherwise it will not work. To do this you have to include the <link rel> tag.

Example

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>New Webpage </head>
    <link rel="stylesheet"     href="style.css">
  </head>
</html>

Make sure when you link your stylesheet to yout HTML that you type the exact name of the CSS file, otherwise it will not load.

Common CSS styles

Style Tags Purpose of the tag
color The colour CSS property specifies the foreground colour of an element's text and text decorations, as well as the currentcolor value.
text-align Changes the positioning of the text on the page, there are multiple options such as center, jutified, left, right, etc.
background Changes the background colour, you can use hex code, rgb and other ways to change the colour. We recommend hex code.
text-size Changes the size of the text anywhere, you can apply it to certain pieces of texts such as paragraphs and headings.
font-family Changes the font in the web-page.
padding Defines the innermost portion of the box model, creating space around an element's content, inside of any defined margins and/or borders.
border Area around the padding that extends from the outer edge of the padding box to the inner edge of the margin
margin The margin is the transparent outermost area that surrounds the border.
max-width Setting the max-width of a block-level element will prevent it from stretching out to the edges of its container.
float The CSS float property specifies how an element should float. The float property is used for positioning and formatting content e.g. let an image float left to the text in a container.
