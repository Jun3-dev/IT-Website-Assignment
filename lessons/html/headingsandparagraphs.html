Headings and Paragraphs

HTML headings and subtitles are headings that you want to display on your webpage, similar to titles on documents and articles.

Example

HTML Headings

HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading (the biggest) and <h6> defines the least important heading. Browsers automatically add whitespace around headings. Most readers skim over large bodies of texts so headers help to structure your texts and provide insight into what different sections are about.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Bigger Headings

Each HTML heading has a default size. However, you can specify the size for any heading with the style attribute, using the CSS font-size property:

<h1 style="font-size:60px;">Heading 1</h1>

HTML Paragraphs

A paragraph always starts on a new line, and is usually a block of text and is defined by the <p> tag. A paragraph always starts on a new line, and browsers automatically add some white space before and after a paragraph.

Example

<p>This is a paragraph</p>

HTML Display

You cannot be sure how HTML will be displayed. Large or small screens, and resized windows will create different results. With HTML, you cannot change the display by adding extra spaces or extra lines in your HTML code. The browser will automatically remove any extra spaces and lines when the page is displayed:

HTML Horizontal Lines

The <hr /> tag defines a thematic break in the HTML page and is most often displayed as a horizontal line. They are generally used to define a new segment in a page and separate content.

Example

<h1>This is a Heading</h1>
<p>This is a paragraph</p>
<hr />

Note that the <hr> tag is a self closing tag (no ending tag).

HTML Line Breaks

The HTML <br /> element defines a line break. Use <br /> to start a new paragraph without making a separate <p> tag.

<p>This is a <br /> paragraph with <br /> multiple breaks </h1>
