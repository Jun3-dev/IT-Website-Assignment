The IFrame Tag

The <iframe>tag specifies an inline frame.

An inline frame is used to embed another document within the current HTML document.

This is useful for integrating things like forums or youtube videos.

E.g:

<iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ" title="Rick Astley's "Never Gonna Give You Up"></iframe>
