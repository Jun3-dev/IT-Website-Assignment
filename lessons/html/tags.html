Basic HTML Tags


HTML Tag Purpose
<html lang="en"> </html> Defines the root of an HTML document and the language
<title> </title> Defines the title (not displayed on the page)
<body> </body> Define's a document's body
<h1> to <h6> Defines HTMl Headings, <h1> being the biggest and os on.
<p> </p> Starts a paragraph, a big body of text in a web page
<img src="" alt="" /> Displays an image here
<i> </i> Set in italics
<br /> Forces a page break
<b> </b> Set in boldface
<ul> </ul> Define's an unorder (bulleted) list
<ol> </ol> Define's an ordered (numbered) list
<li> </li> Brackets an item in an order and unordered list
<a href="url"> </a> Defines a link and it's specified address
<strong> </strong> Sets in bold
<em> </em> Sets in an emphasised text
<hr /> Forces a page rule

HTML Declaration

HTML Files must start with a decleration which will be seen in all HTML files, <!DOCTYPE HTML>

We then follow it with a <html lang="en"> tag. The HTML tag can only have 2 children, and it can only be, the <body> tag and <head> tag

The actual webpage that is visible is shown in the <body> tags.
