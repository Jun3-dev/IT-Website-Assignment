Getting Started

Installing our IDE (Intergarted Development Enviroment)

An IDE is a code editor that allows our code to be excuted, complied, and run. We will be using Visal Studio Code, made by Microsoft. This is the popular IDE in the world, as it is great for everyone. To get started, click on this link and install the correct version for your machine.

Run the installer and follow the prompts

Once in, you will need to install some good extensions to help you code! Use the shortcut control/cmd + shift + F. This should take you to the extensions page. Search for "HTML" and install "HTML Snippets", next search for "Live" and install live server.
