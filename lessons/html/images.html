HTML Images Syntax

The HTML <img> tag is used to embed images onto a web page.

They are not precisely inserted into a website but rather the images are lnked to web pages. The <img> tag merely creates a holding place for the image for the referenced image.

The <img> tag is empty and contains attributes and is an example of a self closing tag (no ending tag). These two required attributes for an image is the:

  • src - specifies a path for the image
  • alt - specifies an alternate path for the image

Syntax

<img src="url" alt="alternatetext" />

The src Attribute

The required src attribute specififies the (URL) to the image.

Note that when a web page loads, the browser retrieves the picture from a web server and places it into the page. As a result, ensure that the picture remains in the same location on the web page; otherwise, your visitors will see a broken link indicator. If the browser cannot find the picture, the broken link symbol and alt text are shown.

Example

<img src="dandelion.jpeg" alt="flower" />

The alt Attribute

If a user is unable to view a picture due to technical difficulties, the alt property is necessary (because of slow connection, an error in the src attribute, or if the user uses a screen reader).

The alt attribute's value should explain the image:

Example

<img src="waves.jpeg" alt="beach" />

If the browser cannot find the image then it will display the alt attribute.

Example

<img src="wrongname" alt="beach" />

A screen reader is a piece of software that interprets HTML code and lets the user to "listen" to the information. People who are visually impaired or learning handicapped might benefit from screen readers.


Image Size - Width and Height

You can use the <style> attribute to specify the image width and height.

Example

<img src="breakfast.jpeg" alt="pancakes" style="width:500px;height:600px;" />

Alternatively, you can also use the width and height attributes:

Example

<img src="breakfast.jpeg" alt="pancakes" width="300" height="400" />

The width and the height attributes specify the width and the height of the image in pixels.

Images in Another Folder

In your HTML coding project if you have a seperate sub-folder for your images you have to include the sub-folder's name in the "src" attribute:

<img src="images/logo.jpeg" alt="HTML 5" width="300" height="400" />
